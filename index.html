Data Analysis Projects

SQL Project

The purpose of this project is to use SQL as an exploratory tool. This project creates a database with its corresponding tables and imports datasets from ourworldindata.org. The analysis of the data is focused on global food security: caloric supply, food expenditure, GDP & food consumption, and protein consumption.

Supply Chain Data Analysis Project

This project aims to use python, pandas and visualization libraries (matplotlib, seaborn, squarify, plotly) in order to analyze the data of a supply chain dataset.

API Project

This project aims to use APIs to obtain information from different sources in order to populate an oscar winner/nominees dataframe.

Book Scraping Project

This project aims to use the library BeautifulSoup on the website www.gatesnotes.com in order to collect the website's book recommendations by year.

Wikipedia Scraping Project

This project aims to use the BeautifulSoup library on the Wikipedia website in order to access the list of sovereign states and iterate through each state to obtain information about the country.

Tableau Project

The goal of this project is to use Tableau to create dashboards using the data from my previous project "Supply chain data analysis". Due to the amount of information and its complexity, the visualizations were divided into five groups: product, customer, orders, shipping & stores.
